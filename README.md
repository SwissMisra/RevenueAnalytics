# RevenueAnalytics
This project uses real company data to construct a data model and identify KPIs for theater performances. As this is data for actual performances and productions, identifying information and proprietary data has been removed.

My goal was to determine the key variables that determine the earned revenue of a "production", i.e. a theater's run of performances of a given work, such as <i>Macbeth</i> as well as the individual performances. In particular, I wanted to examine whether an increase in the average price of a seat in the theater, or "ASP", resulted in increased revenue for a given performance or production.
